# LinguNotify
透過Line Notify隨機傳送每日一個單詞，來學習族語。

## 特點
1.透過apps script每時觸發，可以設定每日固定時間傳送。

2.類型格式可以在sheet增加及code修改。

3.音檔部分目前是套用族語之火合成來使用。

## 已知問題
合成音連結，碰到空格或是其他符號無在手機開啟。
